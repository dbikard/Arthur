# Arthur - Snake Game

A classic Snake game built with vanilla HTML, CSS, and JavaScript. No dependencies, no build step — just open `index.html`.

## How to Play

- **Arrow keys** or **WASD** to move
- **Swipe** on mobile
- Eat the red food to grow and score points
- Don't hit the walls or yourself

## Deploy on GitHub Pages

1. Go to your repository **Settings** > **Pages**
2. Under **Source**, select the branch (e.g. `main`) and folder (`/ (root)`)
3. Click **Save** — your game will be live at `https://<username>.github.io/Arthur/`
